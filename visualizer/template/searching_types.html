Searching Algorithms

Explore the essential searching techniques that are fundamental in Data Structures and Algorithms

Linear Search

Linear Search sequentially checks each element of the list until the target element is found or the list ends.

Binary Search

Binary Search divides the sorted list into halves to find the target element in logarithmic time.

Jump Search

Jump Search examines a sorted list by jumping a fixed number of steps and then performs a linear search.

Interpolation Search

Interpolation Search improves binary search by estimating the position of the target element based on its value.

Exponential Search

Exponential Search locates the range where the target might be and then performs binary search within that range.

Fibonacci Search

Fibonacci Search uses Fibonacci numbers to divide the list and locate the target efficiently.
